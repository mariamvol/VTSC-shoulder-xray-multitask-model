# Input archive format

The application expects a ZIP archive with one study/patient folder inside.

Example:

```text
101.zip
└── 101/
    ├── 1/
    │   ├── IMG-0001-00001.jpg
    │   └── IMG-0001-00002.jpg
    └── 2/
        └── IMG-0003-00001.jpg
```

Supported image formats:
```text
.jpg, .jpeg, .png, .bmp, .tif, .tiff
```
Do not upload personally identifiable medical data unless it has been properly anonymized.
